3-1-4

№ Название тега Краткое описание тега
1 <table> Служит контейнером для элементов, определяющих содержимое таблицы
2 <tr> Cлужит контейнером для создания строки таблицы
3 <td> Предназначен для создания одной ячейки таблицы
4 <th> Определяет заголовок таблицы (ячейка с отцентрованным жирным текстом)
5 <thead> Заголовочная часть таблицы, с помощью данного тега в таблице определяются заголовки столбцов
6 <tbody> Основная часть таблицы, содержит основной набор строк и столбцов которые необходимо вывести браузеру
7 <tfoot> Тег для создания подвала столбцов таблицы, задается после определения заголовков столбцов
8 <caption> Тег для написания заголовка таблицы
9 <col> Задает характеристики одной либо нескольких колонок
10 <colgroup> Определяет группу из одного или нескольких столбцов внутри таблицы
